# OSL Incubator

The OSL Incubator helps open source projects grow from early-stage
proof of concept into sustainable, production-ready projects.

It is a **pre-incubation layer** designed to prepare projects for
ecosystem standards and potential progression into programs such as:

- CNCF
- Apache
- pyOpenSci
- rOpenSci

## Lifecycle

- POC (Proof of Concept)
- Incubation
- Graduation

See: [docs/overview/lifecycle.md](docs/overview/lifecycle.md)

Open an issue using one of the templates:

- POC application
