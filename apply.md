# How to Apply

Applications are handled via GitHub issues.

## Steps

1. Choose your stage (POC / Incubation / Graduation)
2. Open the corresponding issue template
3. Provide required information
4. Wait for reviewer feedback

## Apply here

[https://github.com/esloch/osl-incubator-program/issues](https://github.com/esloch/osl-incubator-program/issues)

---

## What to expect

* Asynchronous review
* Checklist‑based evaluation
* Iterative feedback
