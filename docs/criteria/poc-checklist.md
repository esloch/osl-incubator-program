# POC Checklist

## Purpose

Use this checklist to confirm a project is ready for the sandbox or
experimental stage.

## Scope

This is a lightweight check for basic open source hygiene and a working
prototype.

## Required

- [ ] Public repository
- [ ] OSI-approved license
- [ ] README with purpose and basic usage
- [ ] Working demo, deploy, or equivalent proof of concept
- [ ] CONTRIBUTING.md
- [ ] Code of Conduct
- [ ] At least one maintainer or owner

## Helpful evidence

- [ ] Issue template exists
- [ ] Basic tests exist
- [ ] CI runs on the repository
- [ ] Roadmap or next-step notes exist

## Review rule

If a required item is missing, the project is not ready for incubation.

## Outcome

The project is understandable, usable, and open to contribution.
