# Reviewers

## Purpose

Reviewers validate submitted evidence against the relevant checklist.

## Scope

Use this page for checklist review, follow-up, and due-diligence input.

## Responsibilities

- review submitted evidence
- identify gaps against criteria
- record checklist-based feedback
- request follow-up only when needed
- keep feedback concise and specific

## Review rule

Reviewers should not block a project because of taste, preference, or an
unwritten standard.

## Good feedback

- points to a specific checklist item
- explains what evidence is missing
- suggests the smallest useful next step
