# Lifecycle

OSL uses a 3-stage model:

## POC
Entry stage for early projects.
Focus: usability, clarity, basic OSS hygiene.

## Incubation
Focus: quality, governance, community, maintainability.

## Graduation
Focus: sustainability, adoption, ecosystem readiness.

---

## CNCF alignment

Projects aiming to move toward CNCF should align with:

https://github.com/cncf/toc/blob/main/process/README.md

OSL does not replicate CNCF, but prepares projects for it.
