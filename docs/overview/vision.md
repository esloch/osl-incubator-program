# Vision

## Purpose

OSL Incubator exists to help open source projects mature in a practical,
lightweight, and reviewable way.

## Scope

The program covers the path from an experimental project to a stable
project with repeatable maintenance and visible ownership.

It is intentionally documentation-first and checklist-driven.

## What the program provides

- a clear path from early work to sustainable operation
- visible expectations for contributors and reviewers
- a consistent stage model across projects
- a lightweight due-diligence step for promotion decisions

## What the program does not do

- it does not copy CNCF, pyOpenSci, or rOpenSci policy language
- it does not replace ecosystem-specific review
- it does not require heavy governance for early-stage projects

## Core idea

OSL defines the baseline. External ecosystems can then validate deeper
requirements when a project is ready for them.
